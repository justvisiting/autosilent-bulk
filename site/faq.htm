    In autosilent mode/hours , all sounds except the RINGTONE go silent. What may be going wrong?

  • Make sure the Ringtone is set to OFF in "Silent Profile Settings".
  • If you are getting ringtone and other sounds go off , you may need mobile substrate. Mobilesubstrate is only required to put off ring from contacts which have special ringtone assigned. It is not required otherwise. Try installing mobilesubstrate from Cydia. That should fix ringtone issue.
  • If you have a previous version of autosilent, to optimize battery there is potential delay up to 3 minutes. It might take up to 3 minutes for the phone to go silent. For e.g:There is a meeting from 3:00PM-4:00PM. It might take up to 3:03PM.

    How to make my phone go into silent mode? How do I autosilent my phone?

    There are 2 ways to make the phone to go into Silent mode - Manual or Automatic Mode.

    1. Manual Mode - Set the Manual Mode - ON . Then set the "Ends" time to later time. The manul mode will come out of the Silent mode after the "Ends" time.
    2. Automatic Mode - Set the Automatic mode to ON. You can silent the phone either by setting the weekday night or weekend night time in the app or creating an event in iPhone calendar. Eg: If you want the phone to go into silent mode 3 - 4 PM on Tuesday, you can set the "Weekday Night" to ON, and set the time to 3 - 4 PM. This will make the phone go into silent mode every weekday. Alternatively you can set "iPhone Calender" to ON and then schedule an event on the particular day from 3-4 PM.

    I want the app not to go into silent mode for some events. How do I do that?

    If you have Microsoft exchange based calender (like google calendar or corporate email), the app will go silent only for those events which have "availability" set to "Busy". For other calenders which do not have the "Availability" option (if you don't see "Availability" option in the appointment that means the calendar does not support it), the app currently does not distinguish between such events.

    What is manual mode?

    Manual Mode sets the phone to go silent right now. The phone will automatically come out of silent mode at the specified "Ends" time. If the "Ends" time occurs in the past, the app will not go into silent mode.

    Top

    What is difference between "Automatic Mode" and "Manual Mode"?

    "Automatic Mode" sets the phone to go into silent based on the times set in the Weekday, Weekend schedules and calender events. Manual mode specifies the phone to go into silent mode immediately.

    What is "Disable/Bypass Silent switch"?

  • This option will disable the silent switch on the side of your iphone. In case your silent switch was broken while your phone was in silent mode, this will restore all the original phone sounds.
  • You can use it if the silent switch is broken or loose. Once you enable Bypass silent switch option, you can use the app or AS SB Settings toggle to put the phone into silent mode.

    In the automatic mode, what is "iPhone Calender"?

    When the "iPhone Calender" option is set to ON, autosilent goes in and out of silent mode based on the busy events in your calender. Eg: If you have an event on a tuesday afternoon from 3 - 4 PM , the phone will go silent during that period.
