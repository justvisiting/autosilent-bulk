How to make and sell paid app through Cydia

Unlike iTunes where Apple takes care of all payment, distribution and making sure that only paid app runs on the phone, Cydia does not provide any such mechanism. App developer has to go through complex set of procedures to make sure a) the app is legitimate while running, which requires server side integration, b) integrate with Cydia payment gateway to verify whether user has paid for the license and if there is trial version then possibly integration with another payment system so that license can be bought from app itself.

Following are the steps after developing the app you need to go through to sell paid app:

  1. Integrate with payment gateway. Cydia takes care of the payment when an app is downloaded but does not verify whether an app is a paid app when it is installed or run. Anyone could circumvent payment step and download the app directly from the hosted server. In order to get around the problem, app developer has to make sure that the app is paid when it is run. For this, you need to go through set of steps to integrate with Cydia payment gateway and might take few weeks.
  2. Another problem is how to stop the distribution. Once a user downloads the app, he can assemble all the bits and pieces and distribute for free. App developer needs to make sure the app cannot be redistributed or phone home.
